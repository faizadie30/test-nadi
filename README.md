# How To Start

1. after clone and setup file .env and key project, next step please install project
```bash
composer install
```

2. next running migration
```bash
php artisan migrate
```

3. next running seeder for user administration
```bash
php artisan db:seed --class=UserSeeder
```

4. running
```bash
php artisan serve
```
